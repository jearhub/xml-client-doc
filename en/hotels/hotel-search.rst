Search hotels
#############

Description of XML schema
=========================

XSD-schema response:

- :download:`www/xsd/hotel/HotelSearchRequest.xsd <../../themes/hotelbook/static/xsd/hotel/HotelSearchRequest.xsd>`

- :download:`www/xsd/hotel/HotelSearchResponse.xsd <../../themes/hotelbook/static/xsd/hotel/HotelSearchResponse.xsd>`

Request, HotelSearchRequest
---------------------------

XSD-schema request:

::


    <?xml version="1.0" encoding="utf-8"?>

    <HotelSearchRequest>
      <Request
        cityId="id"
        checkIn="YYYY-MM-DD"
        duration="from 1 to 31"

        [hotelName=""]
        [categoryId="id"]
        [hotelId="id"]
        [locationId="id"]
        [visa="msk|spb"]
        [confirmation="online|onRequest|all"]/>

      [<HotelItems><HotelItemId>...</HotelItemId>...</HotelItems>] - hotel items (hotel facilities)
      <Rooms>
        <Room 
           roomNumber=".." - room count
           adults=".." - adults count
           [children=".."] - children count: 0, 1, ...
           [cots=0|1|2] - cots count
        >
           [<ChildAge>..</ChildAge>] - children ages, from 2 to 18
           [<ChildAge>..</ChildAge>]
           ...
        </Room>
      </Rooms>
    </HotelSearchRequest>

Rooms item
----------

List of required rooms. Maximum number of people in the request may not exceed 9.

| Mandatory item.
| Attributes: no.

Child items:

+------------+-----------------+-------------------+
| **Item**   | **Mandatory**   | **Description**   |
+------------+-----------------+-------------------+
| ``Room``   | yes             | Room parameters   |
+------------+-----------------+-------------------+

Room item
^^^^^^^^^

Room parameters (attributes). If the room has to be a child (children >= 1), it still indicates an element of Child Age.

Mandatory item.

Attributes:

+------------------+-------------------+-----------------+------------------------------------------------------------------+
| **Attribute**    | **Type**          | **Mandatory**   | **Description**                                                  |
+------------------+-------------------+-----------------+------------------------------------------------------------------+
| ``roomNumber``   | numeric           | yes             | number of rooms (min 1)                                          |
+------------------+-------------------+-----------------+------------------------------------------------------------------+
| ``roomSizeId``   | numeric           | no              | id room size ``/xml/room_size``                                  |
+------------------+-------------------+-----------------+------------------------------------------------------------------+
| ``adults``       | numeric, 1 - 4    | yes             | number of rooms (min 1)                                          |
+------------------+-------------------+-----------------+------------------------------------------------------------------+
| ``children``     | numeric 0 - 4     | no              | number of children, 0 (there are no children in room) or 1 - 4   |
+------------------+-------------------+-----------------+------------------------------------------------------------------+
| ``cots``         | numeric 0, 1, 2   | no              | number of cots, 0/1/2                                            |
+------------------+-------------------+-----------------+------------------------------------------------------------------+

 Child items:

+--------------+---------------+----------------------------------------------------------------------------------------------------------------------------------+
| **Item**     | **Mandatory** | **Description**                                                                                                                  |
+==============+===============+==================================================================================================================================+
| ``ChildAge`` | no            | Age of child (2..18). Optional. Items must be present, if the parent item ``Room`` has attribute ``children`` which is not null. |
|              |               | Count of thees items must be equal to the ``children`` value. If age < 2, system will return error                               |
+--------------+---------------+----------------------------------------------------------------------------------------------------------------------------------+

Response, HotelSearchResponse
-----------------------------

XML-schema response:

::


    <?xml version="1.0" encoding="utf-8"?>

    <HotelSearchResponse>
      <HotelSearchRequest>... source request ...</HotelSearchRequest>
      <Errors>
        <Error code="..." description="..."> - list of errors
      </Errors>

      <HotelSearch
        searchId="id"
        countryId="id" countryName=".."
        resourtId="id" resortName=".."
        cityId="id" cityName=".."/>

      <Hotels>
        <Hotel
            resultId="id"
            hotelId="id"
            hotelName=".."
            hotelCatId="id"
            hotelCatName=".."
            confirmation="onRequest|online|inaccessible" -- type of confirmation
            price="orig_price" -- price in foreign currency
            currency=".."
            comparePrice="real_price" -- price in rubles
            duration=".."
            information=".."
            visaMsk="true|false"
            visaSpb="true|false"
            [useNds="true|false"]
            specialOffer="true|false"

            [providerId="id"]
        >
          <Rooms>
            <Room
              roomName="..." 

              roomNumber=".."
              mealId="id"
              mealName="..."
              mealBreakfastId="id"
              mealBreakfastName="..."
              children=".."
              cots="0|1|2"
              sharingBedding="true|false"
            >

              [<ChildAge>2..18</ChildAge>] -- child age
            </Room>
          </Rooms>
          [<Locations>

            <Location id=".." name="..." /> -- list of locations
          </Locations>]
        </Hotel>

      </Hotels>
      [<ProcessedTime>
        [<ProviderTime [providerId="..."] [transferTime="..."] [processTime="..."] />]
        [<DbTime [maxTime="..."] [details="..."] />]
      </ProcessedTime>]
    </HotelSearchResponse>
